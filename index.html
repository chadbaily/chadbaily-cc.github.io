Prior Projects!

UART Connections

With help of documentation, I was able to take a COZIR Ambient 2/10K CO2 + RH/T Sensor and an S8 1% CO2 Sensor and communicate with them via the Raspberry pi's UART connections This was done through the Raspberry pi's USB ports. The data form the sensors is logged and can be stored into a MySQL database located on the Raspberry Pi. This data can then be sent to IoT sites like thingspeak.com to be accessed around the globe.

Helpers Help Me!

During my sophomore year at Stetson University, I was put in charge of managing a team of 9 other students to create a website to log and host volunteer events. The entire website was programmed in C++ using different aids including CGI CC and sqlite3. The website is available for download on GitHub and is completely open-source.

Informaiton Technology Monitoring Wall

During my 2016 summer internship at Virginia 529 College Savings plan, I implemented at physical and virtual server monitoring board. I connected to each physical server through its VMWare monitoring IP, then pulled the information from the website and displayed it using PHP code. Graphs were generated by Google Charts to provide a nice esthetic to the board.

Text Adventure

During my sophomore year at Stetson University, I built a simple text adventure game using C++. The game has hidden rooms, hidden commands, and some hidden items. The game is hosted on GitHub and is available for download to be played.
